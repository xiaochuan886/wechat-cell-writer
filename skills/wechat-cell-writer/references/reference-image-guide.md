# 引用图补充指南

本文档说明如何选择和融入引用图，让图片成为叙事的一部分。

## 图片选择

### 优先级

概念性图片对科普文章最有价值——它们帮助读者理解复杂概念，而不是仅仅装饰页面。

| 优先选择 | 原因 | 搜索关键词 |
|----------|------|------------|
| 概念机制图 | 直观解释原理、流程 | "mechanism diagram", "pathway" |
| 结构示意图 | 建立空间认知 | "structure diagram", "anatomy" |
| 对比图 | 强化差异感知 | "comparison diagram" |

避免选择：纯装饰性图片、与主题关联度低的图片、需要专业背景才能理解的图片。

## 内容融合

引用图的价值在于**增强理解**，而非填充篇幅。图片应该成为叙事的一部分。

### 位置选择

- 紧跟相关段落后（不孤立）
- 不打断重要论述的逻辑链条
- 不在文章开头/结尾孤立放置

### 解释文字

解释文字是连接图片与文章的桥梁。好的解释文字包含三个要素：

1. **承接上文** - 用连接词建立与上一段的联系
2. **解释图片** - 说明图片中的关键信息
3. **呼应主题** - 与文章核心观点建立联系

### 示例

**生硬插入（避免）**：
```markdown
NK细胞很重要。

![NK细胞](imgs/refs/nk-cell.png)

接下来我们讲T细胞...
```
问题：图片与上下文没有联系，解释文字缺失。

**自然融合（推荐）**：
```markdown
NK细胞是我们免疫系统的"第一道防线"。当正常细胞发生癌变时，NK细胞能够快速识别并清除它们。

![NK细胞识别并攻击肿瘤细胞的过程](imgs/refs/nk-cell-mechanism.png)
*上图展示了NK细胞的杀伤机制：通过释放穿孔素在目标细胞膜上打孔，随后注入颗粒酶诱导细胞凋亡。*

这种"先识别、后杀伤"的机制，让NK细胞成为对抗癌症的重要武器...
```
优点：上文铺垫 → 图片 → 解释 → 下文承接，形成完整叙事流。

## 署名规范

引用图必须标注来源，格式简洁：

```markdown
*图源：Wikimedia Commons（CC BY 4.0）。*
```

完整来源信息记录在 `imgs/refs/image-manifest.json`（内部审计用），不放正文。
